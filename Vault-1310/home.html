How to get Started

Log onto github.com, find the Kemp-SUAS repositories and copy the URL of the repository.

Next import the project through eclipse by opening up the eclipse environment. Once its open, navigate to the file tab at the top of the screen, and locate the import function.

Navigate under the git file and click on "projects from git". Then click clone URI, paste the URL at the top and import the project. Once imported, right click on the java project and navigate to configure build path.

Navigate to the libraries and delete the corrupt library. Then click on add external jars and navigate to the library by clicking on assets, then libraries and clicking on the sqlite database library. Finally apply the changes and move to order and export and select the new library and apply changes. Exit out of the build path configurations, open main.java located in renderer and press play to enjoy Fallout TNG.
